# 楽天ROOM 投稿ドラフト

このフォルダには、ブログ記事の自動生成時に同時に作成される「楽天ROOM投稿用の下書き」が保存されます。

## ファイル命名規則

```
{ジャンル}_{YYYYMMDD}.md
例: gadget_20260502.md
```

## 使い方

1. このフォルダ内の最新ファイルを開く
2. スマホの楽天ROOMアプリで投稿（約3分/件）
3. 投稿済みのファイルは削除してもOK

## ジャンルと投稿頻度

| ジャンル | 記事投稿日 |
|---------|---------|
| gadget | 月・木 |
| business | 火・金 |
| investment | 水 |
| travel | 土 |
| gourmet | 日 |
