# 楽天ROOM 投稿ドラフト

このフォルダには、ブログ記事の自動生成時に同時に作成される「楽天ROOM投稿用の下書き」が保存されます。

## ファイル命名規則

```
{ジャンル}_{YYYYMMDD}.md
例: gadget_20260502.md
```

## 使い方（Claude自動投稿）

### ✅ 正しい投稿フロー（楽天市場商品から投稿）

1. 楽天市場で商品を検索: `https://search.rakuten.co.jp/search/mall/キーワード/`
2. 商品ページ（`item.rakuten.co.jp`）を開く
3. ページ内の **「ROOMに投稿」** ボタンをクリック
4. `https://room.rakuten.co.jp/mix/collect?itemcode=...` に遷移
5. コメントを入力して「完了」

### ❌ やってはいけない方法（他人の投稿を転載してしまう）

- `room.rakuten.co.jp/search/item?keyword=...` で検索してコレ！
  → 他ユーザーの投稿を転載になり「〇〇さんの投稿」と表示される

### 手動で投稿する場合

1. このフォルダ内の最新ファイルを開く
2. 楽天市場で検索キーワードの商品を開く
3. 「ROOMに投稿」ボタンから投稿（約3分/件）
4. 投稿済みのファイルは削除してもOK
