## note自動投稿システム（2026年5月構築）

### ファイル構成
| ファイル | 役割 |
|---------|------|
| `src/note_generate.py` | ビジネス記事→note形式に変換（Claude API） |
| `src/note_post.py` | Playwright経由でnote.comに自動投稿 |
| `.github/workflows/note_daily.yml` | 火・金 21:00 JST に自動実行 |
| `data/note_drafts/*.json` | 変換済みドラフト |
| `data/note_posted.json` | 投稿済み管理 |

### 手動実行コマンド
```bash
# ドラフト生成のみ
python src/note_generate.py

# 投稿のみ（ドラフトが必要）
NOTE_EMAIL=xxx NOTE_PASSWORD=xxx python src/note_post.py
```

### noteの記事スタイル（必ず守ること）
- ターゲット：「副業を始めたいけど何から手をつければいいかわからない30代会社員」
- 構成：悩み共感→問題提起→解決策→まとめ＋Novlify誘導CTA
- タイトル：数字入り・50文字以内
- アフィリエイトリンク：楽天・Amazonのみ（それ以外はnote規約違反）

## ③ 楽天ROOM投稿手順（Chrome MCP）

### 重要な技術的注意点（過去の学習から）
> ❌ `javascript_tool` は楽天市場商品ページでブロックされる（Cookie/query string data エラー）  
> ✅ 代わりに `find` + `read_page` でリンクのhrefを取得する

### 手順詳細

#### Step 1: 商品ページを開く
```
navigate( url="https://search.rakuten.co.jp/search/mall/{検索キーワード}/" )
```
- 検索結果から ② の条件に合う商品をクリック
- 商品ページが読み込まれるまで `wait(2秒)` してから次へ

#### Step 2: ROOMに投稿のURLを取得
```
find( query="ROOMに投稿", tabId=XXX )
→ ref_XXX が返ってくる
read_page( tabId=XXX, ref_id="ref_XXX" )
→ href="https://room.rakuten.co.jp/mix?itemcode=...&scid=we_room_upc60" が取得できる
```

#### Step 3: ROOMフォームを開く
```
navigate( url="https://room.rakuten.co.jp/mix?itemcode=...&scid=we_room_upc60", tabId=XXX )
wait(2秒)
```

#### Step 4: コメントを入力する
```
find( query="コメントテキストエリア", tabId=XXX )
→ ref_YYY が返ってくる

form_input( ref="ref_YYY", tabId=XXX, value="コメント本文\n\n#ハッシュタグ1 #ハッシュタグ2 ..." )
```

#### Step 5: 投稿する（完了ボタン）
```
find( query="完了ボタン", tabId=XXX )
→ ref_ZZZ が返ってくる（テキストが "完了" のボタン）

left_click( ref="ref_ZZZ", tabId=XXX )
wait(3秒)
screenshot()
```
→ 「コレ！完了!」画面が表示されれば成功 ✅

## ブログ記事生成ポリシー（generate.py 実行時の必須ルール）

### 🗾 旅行・観光記事は「全国」視点で書く

- **九州・佐賀在住の目線を旅行記事に持ち込まない**
- 出発地は読者によって異なる（東京・大阪・名古屋・北海道など）ため、特定地域発の前提で書かない
- 観光地紹介記事では「その土地の魅力・見どころ・グルメ・アクセス」を全国の読者向けに解説する
- 現在のトピックカバー範囲：京都・北海道・金沢・箱根・鎌倉・江の島・宮古島・屋久島・白川郷・飛騨高山・富士五湖・広島・宮島・全国温泉地など

- generate.py の `_EDITOR_NOTE_PROMPT` に「九州・佐賀在住」という地域条件はない（2026年5月に削除済み）

## サムネイル生成ルール（generate_thumbnail.py）

### ✅ 確定デザイン仕様（2026年5月確定・変更しないこと）

- **背景**: Unsplash写真を明るく活かす（ごく薄いオーバーレイ alpha=55）
- **白い丸角カード**: 縦中央（やや上）に配置（fill alpha=225）、半透明で写真も見える
- **カード内レイアウト（上から）**:
  1. ジャンルバッジ（丸角・ジャンルカラー・左上）
  2. キャッチコピー（ジャンルカラーの小文字・例「もっと旅を楽しもう！」）
  3. タイトル（大きな濃い色の文字・最大2行）
  4. アクセントライン（ジャンルカラーの短い横線）
- **タイトル表示**: `_shorten()` で「：」「｜」「 | 」の前の主題部分だけを使う
  - 例: "新幹線を安く乗る方法：早割・スマートEX…" → "新幹線を安く乗る方法"
  - SEOタイトル全体は記事 frontmatter に残す（サムネイルだけ短縮）
- **ウィドウ防止**: `_fit_lines()` でフォントサイズを6px刻みで下げ、末尾行が最大幅の35%以上になるよう調整
- **最大2行**: 3行目にはみ出さない
- **サイト名**: 右下に小さなグレー文字（novlify.jp）

### ⚠️ やってはいけないこと

- 単色（ソリッドカラー）背景のみのサムネイルに戻さない
- タイトルを「：」以降も含めた長いまま表示しない
- 暗いオーバーレイを強くして写真が見えなくなるデザインにしない
- 3行以上のタイトルを許容しない
- キャッチコピーをカードから外したり削除したりしない

## ⑤ 楽天ROOM公式・人気ブログから学んだ投稿知識

### ハッシュタグの重要ルール（公式より）

| ルール | 内容 |
|--------|------|
| `#オリジナル写真` | **ROOMで最もたくさん検索されているハッシュタグ**。オリジナル写真投稿時は必ず付ける |
| 半角 `#` を使う | 全角 `＃` やシャープ `♯` は検索リンクが機能しない → **必ず半角** |
| ピックアップ欄テーマタグ | 毎週金曜に更新されるテーマに沿ったタグを使うとピックアップ欄に掲載されやすい |

### 季節・イベント別おすすめハッシュタグ

| 時期 | 有効なタグ例 |
|------|------------|
| 1〜3月（新生活） | `#新生活準備` `#入園入学準備` `#春コーデ` `#バレンタイン` |
| 4〜6月（初夏） | `#母の日ギフト` `#梅雨対策` `#UV対策` `#2025春夏` |
| 7〜9月（夏〜秋） | `#暑さ対策` `#防災グッズ` `#秋先取り` `#スーパーSALE` |
| 10〜12月（冬末年） | `#クリスマスギフト` `#年末大掃除` `#2025冬` `#福袋` |
| 通年 | `#ふるさと納税` `#セールで買ったもの` `#新作コスメ` `#ペット用品` |

### コメントを読んでもらうための4つのテクニック

1. **具体的な使い方＋ベネフィット型**  
   例：「スマホと財布がちょうど入るミニバッグ。近所のカフェやお出かけに◎」
   
2. **感情を含める型**  
   例：「見た瞬間に『これだ！』ってなった。シンプルなのに高見えする◎」
   
3. **疑問形で惹きつける型**  
   例：「プロテインって何を選べばいいか迷いませんか？種類を変えてから実感が◎」
   
4. **自分の体験談型**（現在メインで使用中）  
   例：gourmet_20260514 投稿1のホエイプロテインコメント

### 投稿文字数について

| 情報源 | 推奨文字数 |
|--------|----------|
| 過去の実際の投稿経験（優先） | **160〜180文字** |
| 外部人気ブログ（参考） | 80〜150文字がベストとされる |

> 実績ある160〜180文字を基本とするが、「読みやすさ」が最優先。長くなりすぎず、短すぎず。

### 投稿のゴールデンタイム

- **夜9時以降**（夜のスマホチェック時間）
- 通勤時間（朝7〜9時）
- お昼休み（12〜13時）

> 自動化の場合は夜9時〜11時の間に投稿できると理想的。

### ピックアップ欄に掲載されるには

- **毎週金曜更新**のテーマタグを使う
- オリジナル写真 ＋ `#オリジナル写真` タグ
- 「紹介したい商品が正しく伝わる」コメント・写真にする
- 季節・イベントに合ったタグを複数付ける

### その他の注意点

- ハッシュタグは投稿ごとに**季節感のあるものを1〜2個差し替える**と新鮮さが出る
- 商品紹介コメントは「何の商品か正確に伝わること」が最重要（関係のない写真と組み合わせない）
- デメリットも正直に書くと信頼性が向上するが、ROOMでは基本的にポジティブな体験談調で◎

## ⑥ A8ネット・もしもアフィリエイト公式から学んだ記事作成・収益化知識

> このセクションは **ブログ記事を生成・改善する際に参照**する知識です（generate.py の記事品質向上に活用）。

### 【A8キャンパス公式】記事作成7ステップ

#### STEP1: 広告プログラム選定
- サイトのテーマとマッチした商品を選ぶ
- **実店舗で容易に買えるものは避ける** → ネット限定・ニッチな商品が成果に繋がりやすい
- 「ネットで売れるものか？」を必ず確認する

#### STEP2: キーワード選定
| 優先順位 | キーワード例 | 特徴 |
|---------|------------|------|
| 高 | `商品名 口コミ` `商品名 メリット デメリット` `商品名 比較` | **購買意図が高い** → 成果に直結 |
| 中 | `商品名 使い方` `商品名 おすすめ` | 比較検討段階 |
| 低 | `○○とは` `○○店舗` | 情報収集段階 → 成果には繋がりにくい |

- **ロングテール（3語以上）を狙う** → 競合が少なく上位表示しやすい
- ラッコキーワード・Googleキーワードプランナーで検索需要を必ず確認
- キーワードの詰め込みはGoogleペナルティ対象 → 自然な文章が優先

#### STEP3: 記事タイトルの書き方
- **キーワードを冒頭に配置**（検索結果で目立つ）
- **数字を含める**（例：「5選」「30分で」）
- **文字数は30〜40文字以内**（検索結果での省略を防ぐ）
- 記事執筆後にタイトルを見直すと改善しやすい

#### STEP4: 導入文の型
```
① 「こんな悩みありませんか？」→ 共感を誘う問いかけ
② 解決策を提示
③ 記事の概要を1〜2文で
```

#### STEP5〜6: 本文の構成
- 小見出し（H2/H3）を **3〜5個** に整理
- 各見出しで扱うべき視点：使い心地・競合比較・おすすめ理由・ターゲット明確化
- **太字・マーカーで要点を強調**
- 画像は最低3種類（全体像・特徴・細部）
- **主張には客観的な根拠を付ける**（統計データ・行政機関情報など）
- 個人の感想のみでは根拠として不十分

#### STEP7: 記事公開前の確認
- **季節商材は需要ピークの3ヶ月前に公開**（検索エンジンに認識される時間が必要）
- アイキャッチ画像を設定
- SNS共有は「自分の言葉で丁寧に」（自動投稿より個人的な発信が効果的）

### 【A8キャンパス公式】SEO対策の基本3要素

#### 1. 内部SEO
- HTMLタグを正しく階層化（h1→h2→h3）
- 画像の `alt` 属性を設定
- XMLサイトマップを送信

#### 2. コンテンツSEO（最重要）
- **検索エンジンではなく、ユーザーのための情報発信**を徹底
- **E-E-A-T** を意識する

| 要素 | 内容 |
|------|------|
| 専門性（Expertise） | そのジャンルの専門知識・実体験があること |
| 権威性（Authoritativeness） | そのジャンルの専門家として認識されること |
| 信頼性（Trustworthiness） | 事実に基づく情報・運営者情報の明記 |

- **YMYL領域**（金融・健康・法律）は E-E-A-T がさらに厳格に評価される → 要注意

#### 3. 外部SEO
- 良質なコンテンツで自然な被リンクを獲得
- **リンク購入・サテライトサイトはペナルティ対象** → 絶対NG

### 🔴 全ASP共通：絶対に守るべきルール

#### 1. ステマ規制（2023年10月1日〜 景品表示法）

**義務**: アフィリエイトリンクを含む記事・投稿には、必ず「広告であること」を明示する。

| 媒体 | 必須表記と位置 |
|------|-------------|
| ブログ記事 | 記事の**ファーストビュー（上部）**に「PR」「広告」「アフィリエイト広告を利用しています」 |
| サイト全体 | **ヘッダー部分**または全ページ共通部分に表示 |
| SNS投稿 | リンクに**【PR】** または 投稿冒頭に「#PR #広告」 |
| 楽天ROOM | 現時点では楽天ROOMのコレ！投稿は対象外だが、SNS連携投稿時は要PR表記 |

> ❌ NG: PR表記なしに商品を「自分のおすすめ」として紹介する  
> ✅ OK: 記事上部に「本記事はアフィリエイト広告を含みます」と明記

**違反リスク**: 広告主による提携解除・成果報酬キャンセル・A8等ASPからの退会処分

---

#### 2. セルフアフィリエイト禁止（全ASP共通）

- 自分で設置したリンク経由で自分が購入するのは**全ASPで禁止**
- **同居家族・生計を同一にする者**による購入も禁止（楽天アフィリエイト規約より）
- 第三者と示し合わせての相互クリック・購入も禁止

---

#### 3. 不正クリック・不正成果の禁止

- ボット・ソフトウェアによる自動クリック禁止
- 「クリックしてください」「申し込みをお願いします」など報酬目的での依頼禁止
- 架空の個人情報・存在しない情報での申込禁止

### 🟡 法律関連：絶対NGの表現例

#### 景品表示法（優良誤認・有利誤認の禁止）

| NGパターン | 理由 |
|-----------|------|
| 根拠のない「ランキング1位」「No.1」 | 優良誤認 |
| 終了済みセールを「現在開催中」と掲載 | 有利誤認 |
| 実績のない「通常価格5,000円 → 1,980円」 | 有利誤認 |
| 「リピート率80%」など根拠不明の数字 | 優良誤認 |

#### 薬機法・健康増進法（健康食品・化粧品・医療機器）

| NGパターン | ジャンル |
|-----------|---------|
| 「痩せた」「バストアップ」「風邪予防」 | 健康食品 |
| 「シワが消えた」「若返った」 | 化粧品 |
| 「痩せる」「視力回復」 | 雑貨・医療機器 |
| 「抗ガン作用がある」「細胞を活性化」 | 健康食品 |
| 使用前後の写真掲載 | 全般 |

> ⚠️ このブログのジャンル「gourmet」でプロテイン・健康食品を扱う場合は特に注意。**効果・効能を断言する表現は使わない**。

#### 医療法（医療・美容系クリニック等を扱う場合）
- 「必ず治る」「絶対安全」禁止
- ビフォー・アフター写真禁止
- 体験談による広告禁止

#### 著作権法
- 他サイトの文章・画像の無断転載禁止
- 「出典記載すれば引用OK」は誤り。引用には複数の条件が必要
- 芸能人・一般人問わず肖像権に注意
